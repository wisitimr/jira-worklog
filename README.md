
## Prerequisites

> Create .env file

| Environment variable | Description |
| :--- | :--- |
|WORKDIR_IN|path for import jira worklog file|
|WORKDIR_OUT|path for export new worklog file|
|FILE_NAME|file name for jira worklog file|


## Getting start

### Install Packages
```bash
npm install
```

### Generate worklog
```bash
npm run generate
```
